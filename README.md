# рҹҢҹ Overview

AI Resume Analyzer is a modern web application built using **Python** and **Streamlit** that analyzes resumes and provides an ATS (Applicant Tracking System) compatibility score.

The application extracts resume content from PDF and DOCX files, identifies technical skills, detects missing keywords, recommends suitable job roles, and generates a downloadable PDF analysis report.
